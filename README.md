# 🐍 Jeu du Serpent

Bienvenue dans le **Jeu du Serpent** ! Un jeu classique où vous contrôlez un serpent qui grandit en mangeant des pommes. Essayez de ne pas entrer en collision avec les murs ou votre propre corps ! 🍎

## 🎮 Fonctionnalités

- 🐍 Contrôle simple et intuitif du serpent
- 🍏 Le serpent grandit à chaque pomme mangée
- 🚧 Évitez les collisions avec les murs et le serpent lui-même
- 📈 Suivi du score

## 🚀 Installation

1. Clonez ce dépôt :
   ```bash
   git clone https://github.com/votre-utilisateur/snakeGame.git
   ```
2. Naviguez dans le dossier du projet :
   ```bash
   cd snakeGame
   ```
3. Ouvrez le fichier `index.html` dans votre navigateur pour jouer !

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
